* source：检测的目标，可以是单张图片、文件夹、屏幕或者是摄像头：

  * ```python
    python path/to/detect.py --weights yolov5s.pt --source 0              # webcam # 直播软件/电脑摄像头
                                                           img.jpg        # image
                                                           vid.mp4        # video
            										   screen         # screenshot
                                                           path/          # directory
                    								    list.txt	   # list of images
                        							   	list.streams   # list of streams
                                                           'path/*.jpg'   # glob
                                                           'https://youtu.be/Zgi9g1ksQHc'  # YouTube
                                                           'rtsp://example.com/media.mp4'  # RTSP, RTMP, HTTP stream
    ```

  * ```python

    ```

    * ![image](https://github.com/CoderSuHang/TensorRT-Learning-Note/assets/104765251/23874aa6-0673-4c07-8294-f050057199ee)

### 2.2 基于torch.hub的检测方法

* 安装方法按照视频里面教的来。

* 代码示例：

  * ```python
    import torch
    
    # Model 加载本地的模型
    model = torch.hub.load("./", "yolov5s", source="local")
    
    # Images 指定检测图片
    img = "./data/images/zidane.jpg"
    
    # Inference
    results = model(img)
    
    # Results
    results.show()
    ```
